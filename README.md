A structured collection of security checklists maintained by **Valkyri Security**.  
These checklists support internal assessments and help teams validate critical security requirements across wallet architectures and related components.

---

## [Wallet Security Checklist](https://github.com/ValkyriSecurity/checklists/blob/main/wallet-security.md)

Core coverage includes:
- **Private Key Security**  
- **Access Control & Session Handling**  
- **Web Frontend Security**  
- **Domain & Infrastructure Hardening**  
- **Communication Security**  
- **Transfer & Signature Safety**  
- **Policy & Manifest Configuration**

This checklist is designed to enforce consistency, depth, and clarity in the evaluation of wallet systems.

---
